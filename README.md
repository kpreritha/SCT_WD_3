# 🎮 Tic Tac Toe Game

A responsive Tic-Tac-Toe game built using **HTML, CSS, and JavaScript** as part of my **SkillCraft Technology Web Development Internship (Task 03).**

---

## 📌 Features

- Player vs Player Mode
- Player vs Computer Mode
- Responsive Design
- Restart Game
- Win & Draw Detection
- Simple and Clean UI
- Mobile Friendly

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
